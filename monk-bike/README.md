# Monk-Bike Project

## Overview
Monk-Bike is a bike rental application that allows users to easily find and rent bikes for their needs. The application features a user-friendly interface, a booking form, and a variety of bikes to choose from.

## Features
- **Homepage**: The main page includes a header with a type animation, a description of the bike rental service, and a booking form for users to enter their rental details.
- **Second Page**: This page provides additional information about the services offered, customer testimonials, and a call to action to encourage users to rent bikes.
- **Responsive Design**: The application is designed to be responsive and works well on both desktop and mobile devices.

## Installation
To get started with the Monk-Bike application, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/monk-bike.git
   ```
2. Navigate to the project directory:
   ```
   cd monk-bike
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
To run the application, use the following command:
```
npm start
```
This will start the development server and open the application in your default web browser.
